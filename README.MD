# SQL Data Analysis Project

This project contains practical SQL queries designed for **business intelligence** and **data analysis**. The goal of this project is to extract meaningful insights from sales and customer datasets using SQL.

## Project Overview

In this project, I worked on real-world SQL analysis tasks such as:

* Customer segmentation
* Revenue analysis
* Profit tracking
* Monthly sales reporting
* Business performance analysis

The queries help transform raw business data into actionable insights.

## Dataset Used

The project uses sample business datasets such as:

* **sales_data**
* Customer purchase records
* Revenue & profit data
* Transaction dates

## SQL Analysis Performed

### 1. Customer Type Analysis

Analyzed:

* New customers
* Returning customers

Metrics calculated:

* Total orders
* Total revenue
* Total profit

### 2. Monthly Revenue Analysis

Used date formatting and aggregation functions to analyze:

* Month-over-month revenue trends
* Sales growth patterns

### 3. Business Insights Generated

Using SQL, I extracted insights such as:

* Which customer type generates more revenue
* Monthly business performance
* Profit contribution trends
